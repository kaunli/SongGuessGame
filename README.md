## 🎵 SongGuess — Music Trivia Game

SongGuess is an interactive music guessing game where players identify songs based on progressively revealed hints such as album art, lyric fragments, metadata, and audio-style visualizations. The fewer hints used, the higher the score.

## 🎯 Progressive Hint System
- Album art reveal (pixelated progression)
- Lyric fill-in-the-blank hints
- Metadata clues (genre, decade, fun facts)
- Waveform-style audio visualization

## 🧠 Smart Scoring System
- More points for fewer hints used
- Streak-based bonus tracking
- Performance feedback after each round

## 🎮 Multiple Game Modes
- Mixed (all songs)
  - By artist
  - By decade
  - By genre
- Hard mode (fewer hints)
- Speedrun (timed challenges)

## 🏆 Leaderboard System
- Global ranking simulation
- Personal score tracking
- Streak-based comparison

## 🧩 Game Flow
- Select a game mode
- Start a round with 5 songs
- Guess the song using hints
- Earn points based on performance
- View results and leaderboard
